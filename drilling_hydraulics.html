Equations Used — Click to Expand

Rheology — Power Law (API RP 13D, Dual-Model)

PV = R600 − R300 [cp] YP = R300 − PV [lbf/100ft²] n_pipe = 3.32 × log10(R600 / R300) K_pipe = 5.11 × R300 / 511^n_pipe [lbf·sⁿ/100ft²] n_ann = 0.657 × log10(R100 / R3) K_ann = 5.11 × R100 / 170.2^n_ann [lbf·sⁿ/100ft²]

Rheology — Herschel-Bulkley (Yield Power Law)

τy = max(0, 2·R3 − R6) [lbf/100ft²] n_HB = 3.32 × log10((R600−τy)/(R300−τy)) K_HB = (R300 − τy) / 511^n_HB

Velocities (Q gpm, D in inches)

V_pipe (ft/s) = Q / (2.448 · D²) V_ann (ft/s) = Q / (2.448 · (Dh² − Dp²)) V_nozzle (ft/s)= Q / (3.117 · TFA)

Reynolds (Generalized)

μ_eff,p (cp) = 100·K·(96·V/D)^(n−1) · [(3n+1)/(4n)]^n μ_eff,a (cp) = 100·K·(144·V/(Dh−Dp))^(n−1) · [(2n+1)/(3n)]^n Re = 928 · ρ · V · D_h / μ_eff (D_h = D pipe, or Dh−Dp annulus)

Flow Regime

Re_c1 = 3470 − 1370·n (laminar upper) Re_c2 = 4270 − 1370·n (turbulent lower)

Friction Factor / Pressure Loss

Laminar: f = 16 / Re Turbulent: f = a / Re^b, a=(log n + 3.93)/50, b=(1.75 − log n)/7 Trans: linear interp between laminar f(Re_c1) and turb f(Re_c2) ΔP/L (psi/ft) pipe = f · ρ · V² / (25.81 · D) ΔP/L (psi/ft) annulus = f · ρ · V² / (25.81 · (Dh − Dp))

Surface Equipment (Equivalent Pipe Method)

Treat surface eqpt as an equivalent length of 3.826" ID drill pipe Case 1 Le=2600 ft | Case 2 Le=946 ft | Case 3 Le=610 ft | Case 4 Le=424 ft ΔP_s computed using same Power Law pipe equations above

Bit

TFA = Σ (π/4) · (nozzle/32)² [in²] ΔP_bit = ρ · Q² / (10858 · Cd² · TFA²) [psi] V_n = Q / (3.117 · TFA) [ft/s] HHP_bit= ΔP_bit · Q / 1714 [hp] HSI = HHP_bit / (π/4 · D_bit²) IF = 0.01823 · Cd · Q · √(ρ · ΔP_bit) [lbf]

ECD

ECD (ppg) = MW + ΣΔP_ann (surface→depth) / (0.052 · TVD)

Hole Cleaning (Moore slip velocity)

μ_a = PV + 5·YP·(Dh−Dp)/V_ann [cp, apparent] Laminar (Rep ≤ 3): Vs = 4980·ds²·(ρc−ρm)/μa Intermediate: Vs = 174·ds·[(ρc−ρm)/ρm]^0.667 · (ρm/μa)^−0.333 Turbulent (Rep ≥ 300): Vs = 113.4·√(ds·(ρc−ρm)/ρm) Transport Ratio = 1 − Vs/V_ann CCI = (MW · K_ann · V_ann) / 400000

Flow Optimization

For parasitic ΔP ∝ Q^m with m ≈ 1.75: Max HHP at bit: ΔP_bit / ΔP_sys = m/(m+1) ≈ 64% Max Impact Force: ΔP_bit / ΔP_sys = m/(m+2) ≈ 47%
